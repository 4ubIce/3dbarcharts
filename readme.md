* For mocha js console testing:
- install [mocha](https://mochajs.org/) with [npm](https://npmjs.org/) globally:
```
npm install --global mocha
```
- change dir to project dir
- install amd-loader
```
npm install amd-loader
```
- command for testing
```
mocha
```
